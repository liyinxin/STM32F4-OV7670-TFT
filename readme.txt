ALIENTEK OV7670模块摄像头模块,连接关系为:  
		OV7670模块 ------------ STM32开发板
		 OV_D0~D7  ------------  PF0--PF7
		 OV_SCL    ------------  PE5
		 OV_SDA    ------------  PE6
		 OV_VSYNC  ------------  PA8			//同步信号检测IO
         OV_WRST   ------------  PE4		//写指针复位
         OV_WREN   ------------	 PE0		//写入FIFO使能
         OV_RCK	   ------------	 PE1		//读数据时钟
         OV_RRST   ------------  PE3  		//读指针复位
         OV_CS	   ------------	 PE2  		//片选信号(OE)
